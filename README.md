2. Experiential Perspective Prompt
Explanation

An Experiential Perspective Prompt asks the AI to explain something from the viewpoint or experience of a specific person or group. It focuses on feelings, experiences, challenges, and benefits faced by users in real-life situations.

This type of prompt helps in understanding how technology or services affect people in practical situations.

Prompt

“From a college student’s perspective, explain how a smartphone with long battery life, AI-powered camera features, and smooth performance helps in attending online classes, taking notes, communicating with friends, using social media, and entertainment.”

Another Example

“From a teacher’s perspective, describe the challenges and benefits of using smart classrooms and digital learning tools in education.”

3. Everyday Functioning Prompt
Explanation

An Everyday Functioning Prompt explains how a technology, application, or system is used in daily life. It focuses on practical usage and real-world applications that people experience regularly.

This type of prompt is useful for understanding how modern technologies simplify routine tasks and improve productivity.

Prompt

“Describe how AI-powered smartphone features are used in everyday life for photography, video calls, voice assistants, online payments, navigation, social media, and productivity applications.”

Another Example

“Explain how machine learning is used in everyday applications such as YouTube recommendations, Netflix suggestions, Google Maps navigation, and online shopping platforms.”
